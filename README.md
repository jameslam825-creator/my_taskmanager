# MyTaskManager

A comprehensive Android task management application with calendar integration, task tracking, and history logging.

## Features

- ✅ Task Management (Add, Edit, Delete)
- ✅ Calendar View with Task Indicators
- ✅ Task History Log
- ✅ 48-Hour Deadline Notifications
- ✅ Three Status Types: Incomplete, In Progress, Complete
- ✅ Three Priority Levels: Low, Medium, High
- ✅ Material Design UI

## Tech Stack

- **Language**: Java
- **Architecture**: MVVM
- **Database**: Room
- **UI**: Material Design Components
- **Minimum SDK**: 21 (Android 5.0)

## Installation

1. Clone this repository
2. Open in Android Studio
3. Build and run the project
